# Employee Management API

A production-style Employee Management REST API built using Spring Boot.

## 🚀 Tech Stack

- Java 17
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven

## 📌 Features Implemented

- CRUD Operations
- Derived Query Methods
- Pagination (In Progress)
- Sorting (In Progress)
- DTO (In Progress)
- Validation (Upcoming)
- Global Exception Handling (Upcoming)
- Swagger (Upcoming)
- JWT Authentication (Upcoming)
